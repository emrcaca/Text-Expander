# EmR Text Expander

Evrensel metin genişletici kullanıcı betiğidir. Tüm sitelerde yazdığınız belirli kısaltmaları otomatik olarak genişleterek hızlı yazım sağlar.

## Özellikler

* Belirlediğiniz tetikleyicileri otomatik olarak tam metne dönüştürür
* Input, textarea ve contenteditable alanları destekler
* Tüm web sitelerinde çalışır
* Hafif ve hızlıdır

## Kısaltmalar

| Kısaltma | Çıktı                 |
| -------- | --------------------- |
| "1       | Owo h                 |
| "2       | Owo b                 |
| "3       | Owo                   |
| :cp      | Kopyalanan: {{clipb}} |
| :time    | {{time}}              |
| :date    | {{date}}              |
| :ip      | {{ip}}                |

## Kurulum

1. Tarayıcınıza Tampermonkey uzantısını yükleyin
2. Yeni bir kullanıcı betiği oluşturun
3. Depodaki `Userscript.js` içeriğini ekleyin
4. Kaydedip etkinleştirin

## Dosyalar

* `Userscript.js`: Metin genişletme betiğinin tam sürümü
